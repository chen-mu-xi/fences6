## 1. 设计原则

### 1.1 核心设计原则

| 原则 | 描述 | 优先级 |
|------|------|--------|
| **直观性** | 用户无需学习即可理解 Fence 的作用 | P0 |
| **一致性** | 与 Windows 11 Fluent Design 保持一致 | P0 |
| **可定制性** | 用户可完全自定义 Fence 外观 | P1 |
| **性能优先** | UI 响应时间 < 16ms (60fps) | P0 |
| **无干扰** | 不影响用户正常使用桌面 | P1 |

### 1.2 设计风格定位

- **主风格**: Windows 11 Fluent Design
- **配色方案**: 支持浅色/深色模式，适应系统主题
- **圆角**: 8px (Fence 区域), 4px (按钮/控件)
- **阴影**: 轻微投影营造层次感

## 3. Fence 组件设计

### 3.1 Fence 外观结构

```
┌────────────────────────────────────────┐
│ ▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔ │  ← 标题栏 (可拖拽)
│ 📌 常用工具                      _ □ × │  ← 锁定、最小化、关闭
├────────────────────────────────────────┤
│                                        │
│    ┌────┐  ┌────┐  ┌────┐             │

```

### 3.2 Fence 状态

| 状态 | 视觉表现 |
|------|---------|
| **默认** | 标准背景、阴影 Elevation 2 |
| **悬停** | 轻微提亮、阴影 Elevation 3 |
| **拖拽调整大小** | 蓝色边框、高亮 |
| **锁定** | 标题栏显示锁图标 |
| **折叠 (Roll-up)** | 仅显示标题栏 |

### 3.3 Fence 交互

| 操作 | 行为 |
|------|------|
| **创建** | 右键桌面 → "创建 Fence" → 拖拽绘制区域 |
| **移动** | 拖拽标题栏 |
| **调整大小** | 拖拽边缘/角落 |
| **折叠** | 双击标题栏 |
| **删除** | 右键 → "删除" 或拖出所有图标后自动消失 |
| **锁定** | 右键 → "锁定位置" |

## 8. 动画规范

### 8.1 动画时长

| 动画类型 | 时长 | 曲线 |
|----------|------|------|
| **Fence 展开/折叠** | 200ms | Ease-out |
| **Peek 显示/隐藏** | 200ms | Ease-out |
| **Fence 移动** | 100ms | Linear |
| **图标悬停放大** | 150ms | Ease-out |
| **变色龙淡出** | 1000ms | Linear |

## 9. 响应式设计

### 9.1 断点

| 断点 | 屏幕宽度 | 布局调整 |
|------|----------|---------|
| **小** | < 1280px | 单列/双列 Fence |
| **中** | 1280px - 1920px | 双列/三列 Fence |
| **大** | > 1920px | 四列 Fence |

### 9.2 多显示器支持

- 每个显示器独立管理 Fence 布局
- 显示器配置变更时自动调整
- 支持不同 DPI 缩放比例

## 10. 辅助功能

### 10.1 键盘导航

| 快捷键 | 功能 |
|--------|------|
| **Tab** | 在 Fence 间切换 |
| **方向键** | 在图标间移动 |
| **Enter** | 打开选中项目 |
| **Delete** | 从 Fence 移除 |
| **F2** | 重命名 Fence |

### 10.2 屏幕阅读器支持

- 所有控件提供可访问名称
- 焦点状态清晰可见
- 支持高对比度模式
