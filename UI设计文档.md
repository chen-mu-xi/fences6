## 1. 设计原则

### 1.1 核心设计理念

Fences 6 作为桌面增强工具，遵循以下设计原则：

| 原则 | 描述 | 落地场景 |
|------|------|----------|
| **融入而非替代** | Fence 是桌面的一层「透明覆盖」，不改变用户对桌面的认知 | Fence 采用半透明背景，让壁纸隐约可见 |
| **直觉操作** | 右键创建、拖拽组织，学习成本趋近于零 | 所有操作均可通过右键菜单或直接拖拽完成 |
| **性能优先** | 动画流畅不卡顿，后台安静不打扰 | 渲染使用 WPF 虚拟化，文件监控异步非阻塞 |
| **可定制化** | 从外观到行为均可按需调整 | 背景色、透明度、边框、动画速度均可配置 |

### 1.2 色彩体系

```
主色调 (Primary)
├── 浅色主题: #0078D4 (品牌蓝)
├── 深色主题: #60CDFF (亮蓝高光)
└── 强调色: #FFB900 (警告/重要操作)

辅助色 (Secondary)
├── 成功: #107C10
├── 警告: #FF8C00
├── 错误: #D13438
└── 信息: #0078D4

中性色 (Neutral)
├── 浅灰背景: #F3F3F3 / #2D2D2D
├── 中灰边框: #E1E1E1 / #404040
├── 深灰文字: #323130 / #F3F3F3
└── 超深背景: #1F1F1F / #000000
```

### 3.2 围栏边框样式

#### 边框类型

| 类型 | 样式代码 | 视觉效果 |
|------|----------|----------|
| **无边框** | `border: none` | 无边框，适合「融入桌面」风格 |
| **实线** | `border: 1px solid #E1E1E1` | 细边框，最常用 |
| **圆角矩形** | `border-radius: 8px; border: 1px solid #E1E1E1` | 现代柔和风格 |
| **圆角 + 阴影** | `box-shadow: 0 2px 8px rgba(0,0,0,0.15)` | 立体浮起效果 |
| **渐变边框** | `border: 2px solid transparent; background: linear-gradient(...)` | 时尚炫酷 |
| **自定义颜色** | 用户可选任意颜色 | 个性化 |

### 5.1 动画原则

| 原则 | 说明 | 应用 |
|------|------|------|
| **快速响应** | 动画时长 150-300ms | 操作反馈 |
| **流畅自然** | 使用 ease-out 或 ease-in-out | 位置移动 |
| **有目的性** | 动画传达状态变化 | 折叠/展开 |
| **可配置** | 用户可调整速度或禁用 | 性能敏感用户 |
