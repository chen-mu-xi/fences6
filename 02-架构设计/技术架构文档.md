## 1. 技术选型分析

### 1.1 UI 框架对比

| 特性 | WPF | WinUI 3 | 推荐 |
|------|-----|---------|------|
| **目标平台** | Windows 7+ | Windows 10/11 (18362+) | WinUI 3 |
| **设计语言** | 经典 Aero 风格 | Fluent Design / Mica/Acrylic | WinUI 3 |
| **渲染后端** | D3D9/11 | DirectX 12 | WinUI 3 |
| **现代控件** | 需第三方库 | 内置现代化控件 | WinUI 3 |
| **维护状态** | 稳定维护 | 积极演进 | WinUI 3 |
| **企业兼容** | 广泛使用 | 新项目推荐 | WPF (兼容老系统) |

### 1.2 最终技术选型

**推荐方案：渐进式迁移**

| 阶段 | 框架 | 理由 |
|------|------|------|
| **Phase 1-2** | WPF (.NET 6+) | 快速开发，成熟稳定，与现有文档兼容 |
| **Phase 3+** | WinUI 3 (XAML Islands) | 逐步引入 Mica/Acrylic 等现代效果 |
| **最终目标** | WinUI 3 为主 | Windows 11 现代化体验 |

## 3. 关键技术方案

### 3.1 桌面图标拦截与重绘

**方案选择：多方案兼容**

| 方案 | 实现方式 | 优点 | 缺点 | 优先级 |
|------|---------|------|------|--------|
| **A: Shell Namespace Extension** | COM Shell Extension | 原生集成，稳定 | 实现复杂 | P1 |
| **B: Desktop Window Subclassing** | 对 SysListView32 子类化 | 相对简单 | 可能被 Windows 更新破坏 | P0 |
| **C: 自定义桌面覆盖层** | 透明窗口覆盖桌面 | 完全控制 | 需重新实现拖放等 | P2 |

**实现策略：**
```cpp
// 1. 首先尝试 Shell Namespace Extension
// 2. 回退到 Desktop Subclassing
// 3. 最终使用 Overlay 作为降级方案

// 示例：Desktop Window Subclassing
WNDPROC originalWndProc;
LRESULT CALLBACK FenceWndProc(HWND hwnd, UINT msg, WPARAM wParam, LPARAM lParam) {
    switch (msg) {
        case WM_WINDOWPOSCHANGED:
            // 通知 Fence 布局需要更新
            break;
        case WM_DISPLAYCHANGE:
            // 重新计算所有 Fence 位置
            break;
    }
    return CallWindowProc(originalWndProc, hwnd, msg, wParam, lParam);
}
```

```cpp
RegisterHotKey(hwnd, PEEK_HOTKEY_ID, MOD_WIN, VK_SPACE);

// 切换 Peek 模式
void TogglePeek() {
    if (isPeekVisible) {
        // 取消置顶
        SetWindowPos(peekHwnd, HWND_NOTOPMOST, 0, 0, 0, 0, SWP_HIDEWINDOW);
        isPeekVisible = false;
    } else {
        // 设置置顶
        SetWindowPos(peekHwnd, HWND_TOPMOST, 0, 0, 0, 0, 
            SWP_SHOWWINDOW | SWP_NOMOVE | SWP_NOSIZE);
        isPeekVisible = true;
    }
}
```

### 3.3 文件夹 Portal 实时同步

```cpp
// 使用 ReadDirectoryChangesW 监控文件夹
class FileWatcher {
public:
    void StartWatching(const std::wstring& path) {
        dirHandle = CreateFile(
            path.c_str(),
            FILE_LIST_DIRECTORY,
            FILE_SHARE_READ | FILE_SHARE_WRITE | FILE_SHARE_DELETE,
            NULL,
            OPEN_EXISTING,
            FILE_FLAG_BACKUP_SEMANTICS | FILE_FLAG_OVERLAPPED,
            NULL
        );
        
        // 启动异步监控
        std::thread([this]() { MonitorLoop(); }).detach();
    }
    
private:
    void MonitorLoop() {
        FILE_NOTIFY_INFORMATION buffer[1024];
        DWORD bytesReturned;
        
        while (running) {
            if (ReadDirectoryChangesW(
                dirHandle,
                buffer,
                sizeof(buffer),
                TRUE,  // watch subtree
                FILE_NOTIFY_CHANGE_FILE_NAME | 
                FILE_NOTIFY_CHANGE_LAST_WRITE |
                FILE_NOTIFY_CHANGE_DIR_NAME,
                &bytesReturned,
                &overlapped,
                NULL
            )) {
                // 解析变更并通知
                ProcessChanges(buffer);
            }
        }
    }
};
```

### 3.4 多显示器 DPI 感知

```cpp
// 声明 Per-Monitor V2 DPI Awareness
// 在 main() 开始时调用
void SetDpiAwareness() {
    // Windows 10 1703+ 使用 SetProcessDpiAwarenessContext
    DPI_AWARENESS_CONTEXT context = 
        DPI_AWARENESS_CONTEXT_PER_MONITOR_AWARE_V2;
    SetProcessDpiAwarenessContext(context);
}

// 获取指定显示器的 DPI
void UpdateFenceDpi(HWND fenceHwnd, HMONITOR monitor) {
    UINT dpiX, dpiY;
    GetDpiForMonitor(monitor, MDT_EFFECTIVE_DPI, &dpiX, &dpiY);
    
    // 计算 DPI 缩放比例
    double scale = (double)dpiX / 96.0;
    
    // 调整 Fence 位置和大小
    RECT rect = GetFenceRect(fenceHwnd);
    rect.left = (LONG)(rect.left * scale);
    rect.top = (LONG)(rect.top * scale);
    rect.right = (LONG)(rect.right * scale);
    rect.bottom = (LONG)(rect.bottom * scale);
    
    SetWindowPos(fenceHwnd, NULL, rect.left, rect.top,
        rect.right - rect.left, rect.bottom - rect.top,
        SWP_NOZORDER);
}

// 处理 DPI 变更消息
case WM_DPICHANGED: {
    auto newDpi = HIWORD(wParam);
    // 重新布局所有 Fence
    RecalculateAllFences();
    break;
}
```

```cpp
// FenceManager COM Interface
interface IFenceManager : IUnknown {
    // Fence 管理
    HRESULT CreateFence([in] FenceConfig* config, [out] IFence** fence);

    HRESULT GetFences([out] IEnumFences** enumerator);
    HRESULT GetFenceById([in] REFGUID fenceId, [out] IFence** fence);
    
    // 布局管理
    HRESULT ImportLayout([in] LPCWSTR filePath);
    HRESULT ExportLayout([in] LPCWSTR filePath);
    
    // 主题
    HRESULT ApplyTheme([in] LPCWSTR themePath);
    
    // 全局操作
    HRESULT ShowAllFences();
    HRESULT HideAllFences();
    HRESULT TogglePeek();
}

// Individual Fence Interface
interface IFence : IUnknown {
    // 基本属性

    HRESULT GetName([out] BSTR* name);
    HRESULT SetName([in] BSTR name);

    HRESULT GetItems([out] IEnumFenceItems** enumerator);

    // 行为
    HRESULT Collapse();

    HRESULT Lock();
    HRESULT Unlock();
    
    // Portal
    HRESULT SetPortalFolder([in] LPCWSTR folderPath);
    HRESULT GetPortalFolder([out] BSTR* folderPath);
}
```
